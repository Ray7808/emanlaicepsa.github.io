什麼是標頭檔?

wiki

簡單來說，是存放你要使用的功能的地方。

例如，假如你想要使用 cout 這個功能，便需要在程式的開頭加上這行，因為 cout 是 iostream 裡的功能，所以必須 include 後才能使用。

什麼是命名空間?

教學

簡單來說，用來避免那些功能撞名的東西。在 include 後，main 前打上這一段。

若沒有這一行，每次使用 cout 以及其他在 std 命名空間下的功能時，將需要寫成如下形式，十分麻煩。

使用方法

使用#include<標頭檔名>引入要使用的標頭檔。

使用using namespace 名稱; 以使用命名空間。
